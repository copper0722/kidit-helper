# KiDit 小幫手 v0.1.0

給台灣腎臟醫學會 KiDit 網站使用的開源書籤工具。它在病人新增／編輯頁，替縣市、鄉鎮市區、郵遞區號及路街等長下拉選單加上搜尋框。

**護理師不必安裝 plugin、擴充功能或開啟開發者模式。** 由院所管理者在每台護理站電腦設定一次書籤，之後使用者只要登入 KiDit、開啟病人頁、點一下「KiDit 小幫手」。

## 安裝與使用

請開啟公開安裝頁：<https://copper0722.github.io/kidit-helper/>

1. 管理者顯示瀏覽器書籤列，把安裝頁上的「KiDit 小幫手 v0.1.0」按鈕拖到書籤列。
2. 護理師照常登入 KiDit，開啟病人新增或編輯頁。
3. 點書籤列的「KiDit 小幫手」，輸入關鍵字後選擇結果。
4. 確認 KiDit 原本的下拉欄位，再由人工按原有的儲存按鈕。

## 安全與隱私

- 不傳送、不儲存、不記錄任何資料。
- 不載入遠端程式碼，也沒有分析或遙測。
- 不讀寫 cookie，不會自動送出表單或按儲存。
- 只在 `www.kidit-tsn.org.tw` 的病人新增／編輯頁執行；頁面結構不符時停止。
- 原始下拉選單完整保留，KiDit 仍是唯一資料處理端。

這是獨立的社群工具，**不是台灣腎臟醫學會或 KiDit 官方產品**。正式上線前，院所仍應在無真實病人資料的環境完成驗收。請勿在公開 issue 放病人資料、帳密或臨床畫面；安全問題請見 [SECURITY.md](SECURITY.md)。

## 開發

需求：Node.js 18 以上，無第三方套件。

```bash
npm test
```

`src/bookmarklet.js` 是執行碼唯一來源；建置會產生 `dist/` 測試檔與 `docs/` GitHub Pages 安裝頁。授權條款為 [MIT](LICENSE)。
