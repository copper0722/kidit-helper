# 隱私說明

KiDit 小幫手 v0.1.4 完全在目前的瀏覽器頁面內執行。

- 不設置伺服器，不把資料傳到作者、GitHub 或第三方。
- 不使用 `fetch`、XMLHttpRequest、WebSocket、Beacon 或外部程式碼。
- 不使用 localStorage、sessionStorage、IndexedDB 或 cookie。
- 不蒐集分析、錯誤報告、使用紀錄或裝置識別資訊。
- 不自動送出表單；只有使用者自行按下 KiDit 原有儲存按鈕才會送出。

GitHub Pages 僅用來展示安裝說明與提供靜態書籤內容。使用者登入 KiDit 後執行的書籤程式，不會與 GitHub Pages 保持連線。
