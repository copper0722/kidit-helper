管理者：匯入一次，帶自訂圖示

  1. 若已有舊版，先從書籤列刪除「KiDit 小幫手」。
  2. 下載下方的書籤安裝檔。
  3. Chrome 右上角 ⋮ → 書籤和清單 → 匯入書籤和設定，選取剛下載的檔案。
  4. 若書籤出現在「已匯入」資料夾，將「KiDit 小幫手」拖到書籤列。

KiDit 小幫手 v0.1.2
護理師不必處理安裝。 管理者設定完成後，日常仍只要點一下書籤。

護理師：每天只做三步

  1. 照常登入 KiDit，開啟病人的「病史紀錄」頁。
  2. 點書籤列的「KiDit 小幫手」，在「轉入院所」右側搜尋框輸入關鍵字。
  3. 選好後確認 KiDit 原本的下拉欄位，再由人工按原有儲存按鈕。

小幫手不會自動儲存。 原欄位與 KiDit 原流程都會保留。

v0.1.2 支援範圍

目前只支援 KiDit 病史紀錄頁的「轉入院所」搜尋；縣市仍使用 KiDit 原下拉選單。若 KiDit 網址或表單結構不符，工具會顯示提示後停止。

隱私與安全

  • 不傳送或儲存資料
  • 不載入遠端程式碼
  • 無分析、遙測或 cookie
  • 不會自動送出表單
